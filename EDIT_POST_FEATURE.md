# 帖子编辑功能说明

## 功能概述

用户现在可以编辑自己发布的帖子，包括修改标题、内容、分类和图片。

## 使用方法

### 1. 编辑帖子
- 在帖子卡片右上角点击 **三点菜单** (⋮)
- 选择 **"编辑帖子"** 选项（仅作者可见）
- 在弹出的编辑窗口中修改内容
- 点击 **"更新帖子"** 保存修改

### 2. 编辑内容
可以修改以下内容：
- ✏️ **标题** - 修改帖子标题
- 📝 **内容** - 修改帖子正文
- 🏷️ **分类** - 更改帖子分类
- 🖼️ **图片** - 更换、删除或添加图片

### 3. 图片处理
- **保持原图**：不做任何修改
- **更换图片**：点击"上传图片"选择新图片
- **删除图片**：点击图片右上角的 ❌ 删除图片

## 权限说明

- ✅ **作者**：可以编辑和删除自己的帖子
- ✅ **管理员**：可以删除任何帖子（但不能编辑他人帖子）
- ❌ **其他用户**：无法编辑或删除他人帖子

## 技术实现

### 新增函数
在 `lib/supabase.ts` 中新增：
```typescript
export async function updatePost({
  postId,
  title,
  content,
  description,
  category,
  image_url,
  image_ratio,
})
```

### 修改的组件
1. **create-post-modal.tsx**
   - 支持编辑模式（通过 `editPost` prop）
   - 预填充现有数据
   - 区分创建和更新操作

2. **post-card.tsx**
   - 添加"编辑"菜单项
   - 集成编辑模态框
   - 权限控制（仅作者可见）

### 数据库
✅ **无需修改数据库结构**
- 使用现有的 `posts` 表字段
- 通过 `UPDATE` 操作更新数据

## 注意事项

1. **权限验证**：后端会验证用户是否为帖子作者
2. **图片处理**：更换图片时会上传新图片，旧图片保留在存储中
3. **缓存清理**：更新后会自动清理相关缓存
4. **实时更新**：编辑后会触发 `postUpdated` 事件

## 未来优化建议

如果需要更完善的功能，可以考虑：
- 添加 `updated_at` 字段显示最后编辑时间
- 添加 `edit_count` 字段记录编辑次数
- 显示"已编辑"标记
- 保存编辑历史记录

但这些都需要修改数据库结构，当前版本不包含这些功能。
