## 🔧 常见问题和解决方案

### 问题1：快速测试显示错误
**可能的错误信息**：
- `❌ 点赞测试失败: violates foreign key constraint`
- `❌ 评论测试失败: new row violates row-level security policy`

**解决方案**：
在Supabase后台执行以下SQL：

```sql
-- 删除外键约束
ALTER TABLE public.likes DROP CONSTRAINT IF EXISTS likes_user_id_fkey;
ALTER TABLE public.comments DROP CONSTRAINT IF EXISTS comments_user_id_fkey;

-- 临时禁用RLS
ALTER TABLE public.likes DISABLE ROW LEVEL SECURITY;
ALTER TABLE public.comments DISABLE ROW LEVEL SECURITY;
```

### 问题2：应用显示"离线"或"获取数据失败"
**解决方案**：
1. 检查设备网络连接
2. 确认Supabase项目未暂停
3. 重启应用

### 问题3：登录后立即退出
**解决方案**：
1. 清除应用数据（设置 > 应用 > 应用名 > 存储 > 清除数据）
2. 重新安装应用
3. 重新登录
